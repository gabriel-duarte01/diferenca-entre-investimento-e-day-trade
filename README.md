-> Resumo estruturado:

    Day Trade é uma modalidade de esculação financeira - uma aposta - atrelada à bolsa de valores. Há um risco extremamente elevado, porém os retornos podem ser bem robustos. Os investimentos, por outro lado, são algo mais sólidos, pois o investidor realiza a aplicação e num período pre-definido, recebe o valor investido e os dividendos.
